This is a train pack for Subway Builder which currently adds 155 new trains from across the world to Subway Builder using [addTrains](https://github.com/mhmoeller/subwaybuilder-addtrains/tree/V0.12.0-modders.6-Beta) by mhmoller. 

To install, simply download the folder and put it in the mods folder. Make sure to also install addTrains.
